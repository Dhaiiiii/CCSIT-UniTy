
CCSIT UniTy — Home Page Starter (Static Prototype)

How to run locally:
1) Option A (quick):
   - Double-click index.html to open in your browser.

2) Option B (recommended for future backend work):
   - Install XAMPP.
   - Copy this folder to: htdocs/CCSIT-UniTy-home-starter
   - Start Apache, then open: http://localhost/CCSIT-UniTy-home-starter/index.html

Edit tips:
- Update navigation items inside each HTML file's <nav>.
- Add real content where placeholders exist in qa.html, map.html, clubs.html.
- Use styles.css for global styles and script.js for small interactions.

Relative links:
- All links use relative paths (e.g., 'qa.html'), so the pages work when moved together.
